# 🚀 Smart Civic Risk System

A real-time civic issue reporting and risk intelligence platform for smarter and safer cities.

## 🌟 Features
- 📸 Issue Reporting (Image + Location)
- 🔁 Duplicate Detection
- 👍 Community Voting
- ⚠ Risk Scoring System
- 🚗 Route-aware Safety Alerts
- 📊 Dashboard Analytics
- 🗺 Interactive Map Integration
- ⏱ Automated Issue Lifecycle

## 🧠 How It Works
1. Users report issues
2. System detects duplicates
3. Votes increase priority
4. Risk is calculated dynamically
5. Drivers see issues on route
6. Community validates resolution

## 🛠 Tech Stack
- Frontend: Next.js, React, Tailwind
- Backend: Node.js, Express
- Database: MongoDB
- Maps: Leaflet, OpenStreetMap
